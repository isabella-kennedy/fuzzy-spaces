# Install & Setup

## xFigura.ai
This course utilizes **xFigura.ai**, a cloud-based generative design platform. 

### Account Creation
1.  Navigate to [xFigura.ai](https://xfigura.ai).
2.  Sign up using your university email address.
3.  Verify your account.

### Accessing Assignment Workspaces
Do not start from a blank project. Use the "Project Links" provided in each Assignment folder (`assignments/assignment-X/README.md`). These links contain:
*   Pre-loaded model weights tailored for architectural form.
*   Specific parameter presets (guidance scales, seed ranges) optimized for the assignment.

### System Requirements
*   **Browser:** Google Chrome or Edge (Chromium-based) is recommended for WebGL performance.
*   **Internet:** Stable high-speed connection is required for cloud inference.
*   **Hardware:** A mouse with a scroll wheel is highly recommended for navigating the 3D viewport.

## Local Tools (Optional but Recommended)
*   **Adobe Photoshop:** For preparing collage inputs (`.png`, `.jpg`).
*   **Rhino 7 or 8:** For importing and refining `.obj` exports.
*   **Blender:** Free alternative to Rhino for mesh handling.
